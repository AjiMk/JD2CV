# JD2CV

ATS-friendly resume generator built with Next.js.

## Features

- User authentication
- Resume builder with live preview
- Job description integration
- PDF export
- Dark mode support

## Tech Stack

- Next.js 14
- TailwindCSS
- Zustand
- jsPDF

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
