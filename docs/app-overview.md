# Application Overview

JD2CV is a resume-first job search automation app. The user enters profile data once, then the system uses that information to help generate tailored resumes for job descriptions and manage the job search workflow.

## Main User Flow
1. User logs in.
2. User lands on the dashboard and sees progress immediately.
3. User reviews charts, data summaries, and filtered application results.
4. User manages job applications by creating, editing, or deleting records.
5. User keeps personal details in the profile area only.

## Data Model Direction
Job applications should capture company name, role, stage, dates, location, salary, source, notes, and any extra useful fields.

## Dashboard Direction
- Show charts for the job search pipeline.
- Show paginated filtered applications in a table.
- Reuse the existing design system for now.

## Planned Backend
Backend/API support is required, but implementation should follow the design phase.
